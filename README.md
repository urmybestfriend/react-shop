# Shop [![Build Status](https://travis-ci.org/yankouskia/Shop.svg?branch=master)](https://travis-ci.org/yankouskia/Shop)

##Preparation

> Install nodejs

> Install mongodb

## Run project

```sh
npm install
npm run add-db (for adding initial data)
npm start
```

## Run tests

```sh
npm test
```

### Fill db

```sh
npm run add-db
```
